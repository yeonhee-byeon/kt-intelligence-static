Studio

K Studio는 생성형 AI 도입의 복잡한 과정을 간소화한 All-in-One 플랫폼으로
맞춤형 AI를 쉽고 빠르게 구축하고 운영할 수 있습니다.

Customer Value

생성형 AI 도입의 한계를 해결한 플랫폼

No Code 학습 환경과 Drag&Drop 방식으로 누구나 쉽게 생성형 AI를 구현할 수 있으며,
검증 가능한 RAG모델, 다양한 학습 방식, 오픈소스 연동 및 Playground 제공으로 고객 맞춤형 AI 구축이 가능합니다.

  • 누구나 쉽게 만드는 생성형 AI

    No Code와 Drag&Drop 방식으로, 비전문가도 손쉽게 생성형 AI를
    구축하고 활용할 수 있습니다.

  • 유연한 LLM 학습 환경

    Fine-Tuning, PEFT, RLHF 등 다양한 고도화 학습 방식으로 조직
    특화 요구에 유연하게 대응합니다.

  • 간편한 모델 연동과 실험 환경

    HHugging Face에 등록된 오픈소스 모델을 등록, 관리할 수 있는
    카탈로그와 RAG 성능 검증을 위한 Playground를 제공합니다.

  • 검증된 RAG 기반 맞춤형 학습

    국내 RAG 리더보드로 성능을 검증받은 성능 우위 모델 기반으로,
    고객 맞춤형 학습과 운영 사이클을 제공합니다.

K Studio

K Studio의 강력한 기능

복잡한 AI 개발 없이, 대화형 서비스부터 문서 검색, 운영 모니터링까지
필요한 기능을 빠르게 구현하고 안정적으로 운영할 수 있습니다.

  • Multi-Turn 대화 AI 어시스턴트

    다중 발화를 이해하고 맥락을 이어가는 고도화된 대화형 AI로,
    고객 응대와 업무 자동화를 효과적으로 지원합니다.

  • 기업 문서 지능형 검색 엔진

    기업 내 방대한 문서에서 필요한 정보를 빠르고 정확하게
    찾아주는 RAG 기반 검색 기능을 제공합니다.

  • 실시간 AI 모델 성능 모니터링

    배포된 모델의 성능을 실시간으로 확인하고 분석할 수 있는
    대시보드를 통해, 안정적이고 효율적인 운영이 가능합니다.

다양한 최신 LLM 모델 제공

제공 모델 라인업

KT는 자체 개발한 LLM은 물론, 오픈소스 기반 모델과 Microsoft 협력 모델,
외부 3rd Party 모델까지 폭넓게 보유 및 제공합니다.

  • 믿:음2.0

    국내 AI 기술 주권을 위한 한국형 LLM

    • What is

      KT 자체 개발 고객 전용 Private LLM

      고객 데이터 기반 맞춤형 Fine-Tuning 및 Task 최적화

    • Advantages

      KT 내재화 기술을 통한 추론 비용 절감

      모델 경량화 통한 다양한 시스템 유연 적용

  • GPT K 공개 예정

    한국 제도와 문화를 이해하는 Microsoft 협력모델

    • What is

      GPT-4o 기반, 한국 제도·문화 반영

      Microsoft 공동 개발 한국 특화 고성능 모델

    • Advantages

      높은 정확도, 지속적 성능 개선

  • Llama K 공개 예정

    Meta 오픈소스 모델 기반 KT 대규모 학습 데이터셋 적용

    • What is

      다양한 언어·주제 학습 오픈소스 모델

      대규모 데이터 기반 폭넓은 지식 처리

    • Advantages

      복잡한 언어 구조 이해, 자연스러운 대화 생성

      고급 생성 및 다목적 활용에 유리

  • Solar LLM

    3rd Party와 협력한 도메인 특화 모델

    • What is

      도메인 특화형 설계 협력 모델

      Customization 및 Fine-Tuning 최적화 구조

    • Advantages

      대화, 번역, 분석 등 산업별 업무에 특화된 API 제공

      고객 환경에 맞춘 유연한 적용 지원
